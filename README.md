[ 專案簡介 ]
這是一個為「貓管家 SRE」代餵服務設計的單頁式響應網站（Single Page Application / Landing Page）。
網站結合了「工程師」與「寵物照護」的形象，強調透過標準作業程序（SOP）、數據監控（Monitor）與日誌記錄（Log）來提供高品質的貓咪到府服務。

[ 核心特色 ]
1. 響應式設計 (RWD)：適配桌面、平板與手機版面。
2. 品牌識別鮮明：使用暖黃色與深灰色調，呈現溫暖且專業的工程師形象。
3. 互動微動畫：Hero Section 包含漂浮動畫 (Floating animation) 與按鈕懸停效果。
4. 資訊區塊清晰：包含關於我、SRE特色服務、服務區域、價目表與聯絡方式。

[ 網站內容摘要 ]

1. 關於服務者：
   - 背景：大同大學電機系大四學生（30歲重回校園），具備成熟穩重與工程師的細膩。
   - 理念：用工程師的 SOP 標準照顧貓咪。

2. SRE 服務概念：
   - Monitor (監控)：水位、飼料重量測量、拍照記錄。
   - Log (日誌)：紀錄便便形狀、精神狀態、飲食狀況。
   - Availability (可用性)：風雨無阻，準時抵達。

3. 服務地區：
   - 新莊區 (免運費)
   - 三重區 / 板橋區 (車馬費 +$100)

4. 收費標準 (2026版)：
   - 平日服務：$350 / 30分鐘 (含2貓以內)
   - 春節特別服務：$650 / 30分鐘 (除夕至初五)
   - 額外加給：多貓 +$50/隻
